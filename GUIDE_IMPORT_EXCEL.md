# Guide d'utilisation - Import Excel des Salariés

## 📋 Fonctionnalités du Template Excel

Le template Excel d'importation des salariés dispose maintenant de **listes déroulantes** pour faciliter la saisie et éviter les erreurs.

### 🎯 Champs avec Listes Déroulantes

Les champs suivants disposent de listes déroulantes (cliquez sur la flèche ▼ dans la cellule) :

#### 1. **Sexe (M/F)**
- **M** : Masculin
- **F** : Féminin

#### 2. **Situation Familiale**
- Célibataire
- Marié(e)
- Divorcé(e)
- Veuf(ve)

#### 3. **Nature du Contrat**
- **CDI** : Contrat à Durée Indéterminée
- **CDD** : Contrat à Durée Déterminée

#### 4. **Type Regime (Agricole/Non Agricole)**
- **Agricole** : Régime agricole
- **Non Agricole** : Régime non agricole

#### 5. **Mode de Paiement**
- **Virement** : Virement bancaire
- **Espèces** : Paiement en espèces
- **Chèque** : Paiement par chèque

#### 6. **Groupe de Préavis (1-5)**
- **1** : Groupe 1 (préavis minimum)
- **2** : Groupe 2
- **3** : Groupe 3
- **4** : Groupe 4
- **5** : Groupe 5 (préavis maximum)

### 💡 Comment utiliser les listes déroulantes

1. **Cliquez sur la cellule** où vous voulez saisir une valeur
2. **Cliquez sur la flèche ▼** qui apparaît à droite de la cellule
3. **Sélectionnez la valeur** dans la liste qui s'affiche
4. La valeur est automatiquement insérée dans la cellule

### ⚠️ Messages d'erreur

Si vous saisissez une valeur qui n'est pas dans la liste, Excel affichera un message d'erreur :
- **Titre** : "Valeur invalide"
- **Message** : "Veuillez choisir une valeur dans la liste: [options disponibles]"

### 📝 Commentaires d'aide

Passez votre souris sur les en-têtes des colonnes avec listes déroulantes pour voir des commentaires d'aide explicatifs.

### 🔧 Champs spéciaux

#### Codes Bancaires
Les champs suivants sont formatés en texte pour préserver les zéros en début :
- **Code Banque** : Utilisez l'apostrophe `'` avant le code (ex: `'00005`)
- **Code Guichet** : Utilisez l'apostrophe `'` avant le code (ex: `'00081`)
- **Numéro de Compte** : Utilisez l'apostrophe `'` avant le numéro (ex: `'12345678901`)

#### Dates
Utilisez le format **JJ/MM/AAAA** pour toutes les dates :
- Date de Naissance : `15/05/1985`
- Date Embauche : `01/01/2024`
- Date Fin Essai : `31/03/2024`
- CIN Délivré le : `01/01/2020`

### ✅ Avantages des Listes Déroulantes

1. **Réduction des erreurs** : Impossible de saisir une valeur incorrecte
2. **Saisie plus rapide** : Plus besoin de taper, juste sélectionner
3. **Cohérence des données** : Toutes les valeurs sont standardisées
4. **Interface intuitive** : Facile à utiliser pour tous les utilisateurs

### 🚀 Processus d'importation

1. **Téléchargez** le template depuis l'interface
2. **Remplissez** les données en utilisant les listes déroulantes
3. **Sauvegardez** le fichier Excel
4. **Importez** le fichier via l'interface d'importation
5. **Vérifiez** les résultats de l'importation

### 📞 Support

En cas de problème avec l'importation, vérifiez :
- Que toutes les valeurs obligatoires sont remplies
- Que les valeurs des listes déroulantes sont correctement sélectionnées
- Que les formats de date sont respectés
- Que les codes bancaires utilisent l'apostrophe si nécessaire
