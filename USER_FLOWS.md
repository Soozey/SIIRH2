# User Flows

## Protected Payroll Flow

1. RH or comptable prepares payroll period.
2. Inputs arrive from worker data, calendar, HS/HM, absences, primes, and advances.
3. Preview is computed without altering core payroll rules.
4. Payslips and reports are generated.

## Inspection Flow

1. Employee creates complaint or confidential report.
2. Inspector qualifies case and requests evidence if needed.
3. Employer responds with tracked documents.
4. Inspector issues follow-up, recommendation, or compliance action.
5. Alerts and history remain visible per role policy.
