## États Visuels

### État Initial (Aucun Filtre)
```
Option "Tous les salariés" : ● Sélectionnée
Option "Filtrage" : ○ Non sélectionnée
Bouton : "Traiter tous les salariés"
```

### État avec Filtres Activés
```
Option "Tous les salariés" : ○ Non sélectionnée
Option "Filtrage" : ● Sélectionnée
Grille de sélection : Visible
Bouton : "Appliquer X filtre(s)"
```

### État de Chargement
```
Sélecteur : [Spinner] Chargement...
Texte : "Chargement..." avec spinner animé
```

### État Désactivé
```
Sélecteur : Grisé (bg-slate-100)
Texte : "Sélectionnez d'abord un niveau supérieur"
Curseur : not-allowed
```

## Animations

### Ouverture du Modal
```css
@keyframes fade-in {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes zoom-in {
  from { 
    opacity: 0; 
    transform: scale(0.95); 
  }
  to { 
    opacity: 1; 
    transform: scale(1); 
  }
}
```

### Spinner de Chargement
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```

### Transitions
```css
transition-all : 150ms cubic-bezier(0.4, 0, 0.2, 1)
```

## Responsive Design

### Desktop (≥ 768px)
```
- Modal : max-w-4xl (896px)
- Grille : 2 colonnes
- Padding : p-6
```

### Mobile (< 768px)
```
- Modal : max-w-full
- Grille : 1 colonne
- Padding : p-4
```

## Accessibilité

### Contraste
- Texte principal : 4.5:1 minimum
- Texte secondaire : 3:1 minimum
- Boutons : 4.5:1 minimum

### Navigation Clavier
- Tab : Navigation entre les champs
- Enter : Confirmation
- Escape : Fermeture du modal

### ARIA Labels
- Radio buttons : aria-label
- Sélecteurs : aria-label
- Boutons : aria-label
- États : aria-disabled, aria-selected
